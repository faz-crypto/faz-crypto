### 🚲 NYC Citi Bike — Data Engineering Pipeline
Medallion architecture in Databricks: Autoloader ingestion from S3 → Silver layer with **SCD Type 2** modeling and Haversine distance feature engineering → Z-ordered Gold layer powering commuter route dashboards.

`Databricks` `Delta Lake` `PySpark` `AWS S3` `Autoloader` `Medallion Architecture`

---

### ✈️ Flight Data Pipeline — Star Schema Builder
End-to-end pipeline handling Bookings, Flights, Passenger and Airport data. Uses **Lakeflow Declarative Pipelines** (next-gen Delta Live Tables) for cleansing and validation. Gold layer features an auto-generated Star Schema Builder producing SCD and Fact tables.

`Databricks` `Lakeflow DLT` `PySpark` `Star Schema` `SCD`

---

### 🛡️ Insurance Fraud Detection

`Python` `Scikit-learn` `PCA` `Classification`
